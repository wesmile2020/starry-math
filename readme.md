# Starry Math
a math library for webgl/webgpu.

## Installation

```bash
npm install starry-math
```

## Usage

```typescript
import { Vector3 } from 'starry-math';

const v = new Vector3(1, 1, 0);
const v2 = new Vector3(1, 0, 1);

v.add(v2);
console.log(v); // Vector3(2, 1, 1)
```
